## Supported Versions

| Version | Supported          |
| ------- | ------------------ |

## Reporting a Vulnerability

Please visit [here](https://github.com/CadmiumC4/dutty/security/advisories) to report vulnerabilities. **Do NOT** discuss these on public chat platforms or don't use the issues section.
